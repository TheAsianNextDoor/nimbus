# nimbus

## Overview
nimbus is a Twitch chat word cloud generator


## Getting Started with nimbus
1. Clone Repo: https://github.com/TheAsianNextDoor/AutomateChrome.git
2. Download Deps: npm install
3. Start Website: npm start
